# rosette-dfm

Design-for-Manufacturability prediction for photonic layouts: virtual nanofabrication via rasterization, a fabrication prediction model, re-vectorization, and designed-vs-predicted comparison metrics.

- **Depends on:** rosette-core
- **Consumed by:** rosette-python

See the workspace [`AGENTS.md`](../../AGENTS.md) for the crate map, and `cargo doc -p rosette-dfm --open` for the API reference.
