# rosette-core

Core geometry, layout, and cell-hierarchy types for photonic layout. The foundation crate that every other `rosette-*` crate builds on.

- **Depends on:** (no internal crates)
- **Consumed by:** all other `rosette-*` crates

See the workspace [`AGENTS.md`](../../AGENTS.md) for the crate map, and `cargo doc -p rosette-core --open` for the API reference.
