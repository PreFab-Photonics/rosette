# rosette-drc

Design Rule Checking for photonic layouts: minimum/maximum width, spacing, area, edge length, snap-to-grid, enclosure, overlap, edge angles, and self-intersection.

- **Depends on:** rosette-core
- **Consumed by:** rosette-python

See the workspace [`AGENTS.md`](../../AGENTS.md) for the crate map, and `cargo doc -p rosette-drc --open` for the API reference.
