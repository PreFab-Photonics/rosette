Rosette is a photonic integrated circuit design tool. It provides a Python API backed by a high-performance Rust core for designing, simulating, and exporting photonic layouts.

## Installation

```bash
pip install rosette
```

## Quick Start

```python
import rosette

# Create a new design
design = rosette.Design("my_chip")

# Add components and route waveguides
# ...

# Export to GDS
design.to_gds("my_chip.gds")
```
