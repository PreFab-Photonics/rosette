Rosette is pre-1.0. Function signatures, class interfaces, and behavior may change between releases without notice.

```python
from rosette import *

# Create a simple cell
cell = Cell("my_design")
cell.add_polygon(Polygon.rect(Point.origin(), 10, 5), Layer(1, 0))
write_gds("output.gds", cell)

# Route between ports
route = Route(Layer(1, 0), width=0.5, bend_radius=5.0)
route.start_at_port(port_a)
route.to(50, 0)
route.end_at_port(port_b)
cell = route.to_cell("my_route")
```

In user projects (after `rosette init`), components are local:
```python
from components import mmi_1x2, ring, grating_coupler
```

For library development, import from `rosette.components`:
```python
from rosette.components import mmi_1x2, ring, grating_coupler
```

Default layer definitions used as fallback when `rosette.toml` has no `[layers]`
section. Contains two layers: silicon (1/0), text (10/0).

### I/O

<PyFunction name={"read_gds"} type={"(path) -> Library"}>

Read a GDS file and return a Library.

<Callout title={"Example"} type={"example"}>

```python
lib = read_gds("input.gds")
for cell in lib.cells():
    print(cell.name)
```

</Callout>

<div>

<PyParameter name={"path"} type={"str | Path"}>

Path to the GDS file.

</PyParameter>

</div>

<PyFunctionReturn type={"Library"}>

A Library containing all cells from the GDS file.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"write_gds"} type={"(path, design, cells=None, *, quiet=False, verbose=False) -> None"}>

Write a Cell or Library to a GDS file.

When writing a Cell built using Instance references (via `cell.at()`),
child cells are automatically collected. A build summary is printed
to stderr by default.

<Callout title={"Example"} type={"example"}>

```python
# Auto-collection (recommended):
top = Cell("top")
top.add_ref(gc_cell.at(0, 0))
write_gds("output.gds", top)

# Suppress output for batch processing:
write_gds("output.gds", top, quiet=True)
```

</Callout>

<div>

<PyParameter name={"path"} type={"str | Path"}>

Output file path.

</PyParameter>
<PyParameter name={"design"} type={"Cell | Library"}>

Cell or Library to write.

</PyParameter>
<PyParameter name={"cells"} type={"list[Cell] | None"} value={"None"}>

Optional list of child cells. If `None` and design is a Cell,
child cells are auto-collected from Instance references.

</PyParameter>
<PyParameter name={"quiet"} type={"bool"} value={"False"}>

If `True`, suppress the build summary.

</PyParameter>
<PyParameter name={"verbose"} type={"bool"} value={"False"}>

If `True`, print detailed build info including port positions.

</PyParameter>

</div>

<PyFunctionReturn type={"None"} />

</PyFunction>

### Configuration

<PyFunction name={"load_layer_map"} type={"(config_path=None) -> LayerMap"}>

Load layer definitions from `rosette.toml`.

Reads the `[layers]` section which maps semantic names to GDS layer
numbers with optional display properties (color, fill, opacity).

<Callout title={"Example"} type={"example"}>

```python
layers = load_layer_map()
layers.silicon.layer   # Layer(1, 0)
layers.silicon.color   # "#ff69b4"
```

</Callout>

<div>

<PyParameter name={"config_path"} type={"str | Path | None"} value={"None"}>

Optional explicit path to `rosette.toml`. If `None`, searches
from current directory upward.

</PyParameter>

</div>

<PyFunctionReturn type={"LayerMap"}>

LayerMap with named layer access.

</PyFunctionReturn>

</PyFunction>

### DRC

<PyFunction name={"load_drc_rules"} type={"(config_path=None) -> DrcRules"}>

Load DRC rules from `rosette.toml`.

Searches for `rosette.toml` in the current directory and parent directories.
Rules are defined per-layer in the `[drc.layers]` section (supporting
`min_width`, `min_spacing`, `min_area`, `angles`, `min_edge_length`,
`max_width`, `no_self_intersection`, and `no_overlap`), and inter-layer rules in the
`[[drc.rules]]` array.

<Callout title={"Example"} type={"example"}>

```python
rules = load_drc_rules()
result = run_drc(cell, rules)
```

</Callout>

<div>

<PyParameter name={"config_path"} type={"str | Path | None"} value={"None"}>

Optional explicit path to `rosette.toml`. If `None`, searches
from current directory upward.

</PyParameter>

</div>

<PyFunctionReturn type={"DrcRules"}>

DrcRules built from the configuration.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"run_drc"} type={"(cell, rules, library=None) -> DrcResult"}>

Run DRC on a cell.

<Callout title={"Example"} type={"example"}>

```python
rules = load_drc_rules()
result = run_drc(cell, rules)
if result.passed:
    print("DRC passed!")
else:
    for v in result.violations:
        print(f"  {v.message}")
```

</Callout>

<div>

<PyParameter name={"cell"} type={"Cell"}>

The cell to check.

</PyParameter>
<PyParameter name={"rules"} type={"DrcRules"}>

DRC rules to apply.

</PyParameter>
<PyParameter name={"library"} type={"Library | None"} value={"None"}>

Library containing referenced cells (required if cell has refs).

</PyParameter>

</div>

<PyFunctionReturn type={"DrcResult"}>

DrcResult with violations and statistics.

</PyFunctionReturn>

</PyFunction>

### Design Checks

<PyFunction name={"load_checks_config"} type={"(config_path=None) -> ChecksConfig"}>

Load design checks config from `rosette.toml`.

Reads the `[checks]` section. If absent, returns a `ChecksConfig`
with sensible defaults.

<Callout title={"Example"} type={"example"}>

```python
config = load_checks_config()
result = run_checks(cell, config)
```

</Callout>

<div>

<PyParameter name={"config_path"} type={"str | Path | None"} value={"None"}>

Optional explicit path to `rosette.toml`. If `None`, searches
from current directory upward.

</PyParameter>

</div>

<PyFunctionReturn type={"ChecksConfig"}>

ChecksConfig built from the configuration.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"run_checks"} type={"(cell, config=None, library=None) -> ChecksResult"}>

Run design checks on a cell.

Runs all design checks: connectivity (unconnected ports, width/angle
mismatch) and bend radius (below minimum, auto-reduced). Ports on
the top-level cell are treated as external I/O and are not flagged
as unconnected.

When called with a Cell built using Instance references, child cells
are automatically collected into a Library for hierarchy resolution.

<Callout title={"Example"} type={"example"}>

```python
config = ChecksConfig(min_bend_radius=5.0)
result = run_checks(cell, config)
if not result.passed:
    for v in result.violations:
        print(f"  {v.message}")
```

</Callout>

<div>

<PyParameter name={"cell"} type={"Cell"}>

The cell to check.

</PyParameter>
<PyParameter name={"config"} type={"ChecksConfig | None"} value={"None"}>

Checks configuration. Defaults to `ChecksConfig()`.

</PyParameter>
<PyParameter name={"library"} type={"Library | None"} value={"None"}>

Library containing referenced cells. If `None` and cell has Instance
tracking, a Library is auto-built.

</PyParameter>

</div>

<PyFunctionReturn type={"ChecksResult"}>

ChecksResult with violations and statistics.

</PyFunctionReturn>

</PyFunction>

### DFM

<PyFunction name={"load_dfm_config"} type={"(config_path=None) -> tuple[DfmConfig, GaussianModel, list[Layer]]"}>

Load DFM configuration from `rosette.toml`.

Reads the `[dfm]` section which configures the virtual nanofabrication
prediction tool.

<Callout title={"Example"} type={"example"}>

```python
config, model, layers = load_dfm_config()
result = run_dfm(cell, layers=layers, model=model, config=config)
```

</Callout>

<div>

<PyParameter name={"config_path"} type={"str | Path | None"} value={"None"}>

Optional explicit path to `rosette.toml`. If `None`, searches
from current directory upward.

</PyParameter>

</div>

<PyFunctionReturn type={"tuple[DfmConfig, GaussianModel, list[Layer]]"}>

Tuple of (config, model, layers) where layers is the list
of layers to predict (empty list means all layers).

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"run_dfm"} type={"(cell, layers, model=None, config=None, library=None) -> DfmResult"}>

Run DFM prediction on a cell.

Rasterizes each specified layer, applies the fabrication prediction model,
and extracts contour polygons representing the predicted fabricated geometry.

<Callout title={"Example"} type={"example"}>

```python
result = run_dfm(cell, layers=[Layer(1, 0)])
for lp in result.layers:
    print(f"  Layer {lp.layer}: {lp.input_polygon_count} -> {lp.predicted_polygon_count}")
```

</Callout>

<div>

<PyParameter name={"cell"} type={"Cell"}>

The cell to predict.

</PyParameter>
<PyParameter name={"layers"} type={"list[Layer]"}>

Layers to process.

</PyParameter>
<PyParameter name={"model"} type={"GaussianModel | None"} value={"None"}>

The prediction model. Defaults to `GaussianModel(sigma=0.08)`.

</PyParameter>
<PyParameter name={"config"} type={"DfmConfig | None"} value={"None"}>

DFM configuration. Defaults to `DfmConfig()`.

</PyParameter>
<PyParameter name={"library"} type={"Library | None"} value={"None"}>

Library containing referenced cells (required if cell has refs).

</PyParameter>

</div>

<PyFunctionReturn type={"DfmResult"}>

DfmResult with per-layer predictions and statistics.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"add_dfm_predictions"} type={"(cell, result, datatype_offset=100) -> None"}>

Add DFM predicted polygons to a cell on offset layers.

For each layer in the DFM result, adds the predicted polygons to the cell
on a new layer with the same number but datatype offset by `datatype_offset`.
This makes it easy to visualize designed vs predicted geometry side-by-side.

<Callout title={"Example"} type={"example"}>

```python
result = run_dfm(cell, layers=[Layer(1, 0)])
add_dfm_predictions(cell, result)
write_gds("output.gds", cell)
# Layer 1/0 = designed, Layer 1/100 = predicted
```

</Callout>

<div>

<PyParameter name={"cell"} type={"Cell"}>

The cell to add predicted polygons to (modified in-place).

</PyParameter>
<PyParameter name={"result"} type={"DfmResult"}>

DFM prediction result from `run_dfm`.

</PyParameter>
<PyParameter name={"datatype_offset"} type={"int"} value={"100"}>

Offset added to the original datatype.

</PyParameter>

</div>

<PyFunctionReturn type={"None"} />

</PyFunction>

<PyFunction name={"offset_polygon"} type={"(centerline, width) -> Polygon"}>

Create a polygon from a centerline and uniform width.

The polygon is created by offsetting the centerline perpendicular to the
path direction at each point, forming a "ribbon" shape.

<div>

<PyParameter name={"centerline"} type={"list[Point]"}>

List of points defining the centerline path (minimum 2 points).

</PyParameter>
<PyParameter name={"width"} type={"float"}>

Width of the polygon.

</PyParameter>

</div>

<PyFunctionReturn type={"Polygon"}>

A closed polygon.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"offset_polygon_varying"} type={"(centerline, widths) -> Polygon"}>

Create a polygon from a centerline with varying width.

Similar to `offset_polygon`, but allows specifying a different width at each
centerline point for tapered or variable-width shapes.

<div>

<PyParameter name={"centerline"} type={"list[Point]"}>

List of points defining the centerline path.

</PyParameter>
<PyParameter name={"widths"} type={"list[float]"}>

Width at each centerline point (must have same length as centerline).

</PyParameter>

</div>

<PyFunctionReturn type={"Polygon"}>

A closed polygon.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"path_length"} type={"(points) -> float"}>

Calculate the total length of a polyline path.

<div>

<PyParameter name={"points"} type={"list[Point]"}>

List of points defining the path.

</PyParameter>

</div>

<PyFunctionReturn type={"float"}>

Sum of distances between consecutive points.

</PyFunctionReturn>

</PyFunction>

<PyFunction name={"fresnel_c"} type={"(t) -> float"}>

Fresnel cosine integral C(t).

Used for generating Euler (clothoid) spiral bends.
C(t) = integral from 0 to t of cos(pi/2 * u^2) du.

<div>

<PyParameter name={"t"} type={"float"}>

Upper limit of integration.

</PyParameter>

</div>

<PyFunctionReturn type={"float"} />

</PyFunction>

<PyFunction name={"fresnel_s"} type={"(t) -> float"}>

Fresnel sine integral S(t).

Used for generating Euler (clothoid) spiral bends.
S(t) = integral from 0 to t of sin(pi/2 * u^2) du.

<div>

<PyParameter name={"t"} type={"float"}>

Upper limit of integration.

</PyParameter>

</div>
