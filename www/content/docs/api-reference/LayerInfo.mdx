---
title: LayerInfo
---

A single layer definition with metadata.

Combines the GDS layer identity (number, datatype) with display
properties (color, fill, opacity) and a human-readable description.

Use `info.layer` to get the underlying Layer for rosette APIs:

```python
layers = load_layer_map()
cell.add_polygon(poly, layers.silicon.layer)
```

## Attributes

<PyAttribute name={"name"} type={"str"}>

Semantic name (e.g., `"silicon"`, `"text"`).

</PyAttribute>

<PyAttribute name={"layer"} type={"Layer"}>

The underlying `Layer(number, datatype)`.

</PyAttribute>

<PyAttribute name={"color"} type={"str"}>

Hex color string (e.g., `"#ff69b4"`).

</PyAttribute>

<PyAttribute name={"fill"} type={"str"}>

Fill pattern: `"solid"`, `"hatched"`, `"crosshatched"`, or `"dotted"`.

</PyAttribute>

<PyAttribute name={"opacity"} type={"float"}>

Fill opacity from 0.0 to 1.0.

</PyAttribute>

<PyAttribute name={"description"} type={"str"}>

Human-readable description.

</PyAttribute>

<PyAttribute name={"number"} type={"int"}>

GDS layer number (shortcut for `layer.number`).

</PyAttribute>

<PyAttribute name={"datatype"} type={"int"}>

GDS datatype (shortcut for `layer.datatype`).

</PyAttribute>
